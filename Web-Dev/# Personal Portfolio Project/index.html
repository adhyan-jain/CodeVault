Colorful T-Rex Runner Game

A vibrant, browser-based remake of the classic T-Rex runner game with a modern twist. Built using HTML, CSS, and JavaScript, it features smooth animations, colorful visuals, dynamic obstacle generation, and real-time score tracking. Optimized for both desktop and mobile, the game supports swipe gestures like “swipe up to jump” and can be installed as a standalone mobile app for an immersive, full-screen experience.

CATAPULT

A physics-based shooting game where you launch a hexagon to knock down stacked blocks. Built with HTML, CSS, and JavaScript, the game features mouse-based aiming, smooth launch mechanics, and real-time score tracking. Hit blocks disappear on impact, and the hexagon resets to its position with the spacebar. Simple, interactive, and fun to play!

NODE MAILER

This project is an automated email sender using Node.js, Express.js, and Nodemailer, with a frontend built in HTML, CSS, and JavaScript. It reads recipient data from an Excel sheet and uses a cron job to schedule email sending. Dotenv is used to securely manage environment variables.
